=========
Iterators
=========

Template class 
``flattened2d`` supports forward iterators only. 

The following table provides additional information on the
members of this template class.

==========================================================================================

------------------------------------------------------------------------------------------

  local copies.
------------------------------------------------------------------------------------------

  \
  **Returns**:

  the set of local copies.
------------------------------------------------------------------------------------------

  \
  **Returns**: 
  ``const_iterator`` pointing to the end of the
  set of local copies.
------------------------------------------------------------------------------------------
