==================
continue_msg Class
==================


Summary
-------

An empty class that represent a continue message. An object of this
class is used to indicate that the sender has completed.

Syntax
------

.. code:: cpp

   class continue_msg;


Header
------

.. code:: cpp

   #include "tbb/flow_graph.h"


Members
-------

.. code:: cpp

   namespace tbb { namespace flow { class continue_msg {}; } }
